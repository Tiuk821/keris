Mulai AR

Website Resmi TosanAji.id

Proyek ini merupakan bagian dari penelitian tugas akhir yang mengembangkan media pembelajaran interaktif berbasis Augmented Reality untuk memperkenalkan berbagai jenis keris tradisional Indonesia. Aplikasi ini menggunakan teknologi WebAR sehingga dapat diakses langsung melalui browser tanpa perlu menginstal aplikasi tambahan.

Situs Resmi Kami

Petunjuk Penggunaan

  1. 1. Siapkan marker keris yang telah disediakan.
  2. 2. Klik tombol Mulai AR di halaman ini.
  3. 3. Izinkan akses kamera pada perangkat Anda.
  4. 4. Arahkan kamera ke marker untuk melihat model 3D keris secara langsung.
  5. 5. Geser jari untuk memperbesar atau memutar tampilan jika tersedia.
